# Uniswap Foundation Hackathon (Notes)

## Overview
Uniswap v4 is an extensible AMM protocol designed to support custom market structures using shared, secure liquidity. The Foundation is calling on developers to build on v4 and explore how liquidity, trading, and financial infrastructure can evolve onchain.
